## 数独游戏

一个基于 Vue 3 + Vite 构建的现代化数独游戏 Web 应用，采用自然绿色配色与温暖金色点缀的精致界面设计。

**技术栈**：Vue 3 · Vite · Canvas API · localStorage

**核心功能**：
- 三档难度（简单 / 中等 / 困难）
- 关联单元格智能高亮（同行、同列、同宫格）
- 基于 Canvas 的手写标记功能，支持多色笔记
- 游戏进度自动保存，随时继续
- 完成谜题后的庆祝动画（彩纸纷飞 + 奖杯弹跳）

**在线体验**：[sudoku.hp110.cn](https://sudoku.hp110.cn/)

**源码**：[GitHub → zhoujarvy/sudoku](https://github.com/zhoujarvy/sudoku)

## 算术练习题生成器

一个轻量实用的算术练习题生成工具，支持 A4 格式精确预览和打印输出，适合家长和老师快速生成练习题。

**技术栈**：Vue 3 · 纯 CSS · 单文件 SPA · Cloudflare Pages

**核心功能**：
- 五种运算类型（加、减、乘、除、加减混合）
- 可自定义数字范围和题目数量
- A4 格式精确预览，支持学生信息栏和演算空间
- 智能分页计算，打印效果与预览一致
- 响应式布局，移动端友好

**在线体验**：[math.hp110.cn](https://math.hp110.cn/)

**源码**：[GitHub → zhoujarvy/math-practice](https://github.com/zhoujarvy/math-practice)
